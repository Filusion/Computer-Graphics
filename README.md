## Projects

### **Homework 1 – Triangle Rasterization**
A simple triangle rasterizer that reads vertex data from a JSON file and draws triangles on the canvas. You can drop a JSON file containing triangle definitions, and the application will render them automatically.

### **Homework 2 – Bezier Curve Drawing Application**
An interactive app for drawing and editing Bezier curves. Supports multiple curves and real-time control point manipulation.

### **Homework 3 – Illumination Models**
A demonstration of different lighting models in computer graphics, including:
- Lambert  
- Phong  
- Blinn-Phong  
- Burley  

### **Super Palla Rossa**
A 3D browser game made with WebGPU. Collect coins, stars, and defeat enemies while exploring a colorful 3D environment inspired by classic platformers.
